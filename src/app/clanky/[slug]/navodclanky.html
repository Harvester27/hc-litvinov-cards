🚀 Jak přidat nový článek

1 Otevřete soubor articleData.js

Najděte pole articles a přidejte nový objekt na začátek pole.

2 Vytvořte nový objekt článku

Použijte následující strukturu:

{ // Unikátní ID článku (malá písmena, pomlčky místo mezer) id: 'nazev-clanku-2025', // Titulek článku zobrazený na webu title: 'Název vašeho článku', // URL slug (stejný jako ID) slug: 'nazev-clanku-2025', // Krátký popis (max 160 znaků) excerpt: 'Krátký popis článku...', // Kategorie: 'Turnaje' | 'Zápasy' | 'Klub' | 'Mládež' category: 'Turnaje', author: { name: 'Tým HC Lancers', id: 'hc-lancers' }, // Datum ve formátu: 'DD. měsíc YYYY' date: '27. srpna 2025', // JavaScript Date objekt publishedAt: new Date('2025-08-27T10:00:00'), // Emoji nebo null image: '🏒', // Cesta k obrázku nebo null featuredImage: '/images/clanky/obrazek.jpg', featured: true, views: 0, likes: 0, // HTML obsah článku content: `...HTML obsah...`, allowComments: true, // Tagy pro SEO tags: ['turnaj', 'hokej', 'Litvínov'] }

✏️ Psaní obsahu článku

Základní HTML struktura:

content: ` <div class="space-y-4"> <!-- Úvodní box --> <div class="bg-gradient-to-r from-red-50 to-gray-50 rounded-xl p-6 mb-6"> <p class="text-xl font-bold text-gray-800 mb-2"> Titulek boxu </p> <p class="text-gray-600"> Informace v boxu </p> </div> <!-- Odstavec textu --> <p> Text článku s <strong>tučným textem</strong>. </p> <!-- Nadpis --> <h3 class="text-xl font-bold text-gray-900 mt-6 mb-3">Nadpis sekce</h3> <!-- Obrázek --> <div class="rounded-lg overflow-hidden shadow-md my-6"> <img src="/images/clanky/foto.jpg" alt="Popis" class="w-full h-auto" /> <p class="text-xs text-gray-500 text-center p-2 bg-gray-50">Popisek</p> </div> </div> `

🎨 CSS třídy pro styling

Boxy a upozornění:

  • bg-green-50 border-green-600 - Úspěch/výhra
  • bg-red-50 border-red-600 - Prohra/varování
  • bg-blue-50 border-blue-600 - Informace
  • bg-amber-50 border-amber-600 - Tip/poznámka

Text:

  • text-xl font-bold - Nadpis sekce
  • text-lg - Větší text
  • text-sm text-gray-600 - Menší popisný text
  • font-bold text-red-600 - Důležitý text

📸 Přidávání obrázků

1 Nahrajte obrázky do složky

/public/images/clanky/ nebo vytvořte podsložku

2 Použijte v článku
<img src="/images/clanky/nazev-obrazku.jpg" alt="Popis obrázku" />
⚠️ Formáty: Používejte JPG/JPEG pro fotky, PNG pro loga, SVG pro ikony

✅ Checklist před publikováním

  • ☐ Unikátní ID a slug
  • ☐ Správná kategorie
  • ☐ Datum publikace
  • ☐ Excerpt do 160 znaků
  • ☐ Všechny obrázky nahrány
  • ☐ HTML obsah zkontrolován
  • ☐ Tagy pro SEO
  • ☐ allowComments: true
✅ Hotovo! Po uložení souboru articleData.js bude článek automaticky dostupný na webu.

🆘 Časté problémy

  • Článek se nezobrazuje: Zkontrolujte, že máte správně uzavřené uvozovky a závorky
  • Obrázky nefungují: Ověřte cestu - začíná lomítkem /images/...
  • Odkazy na hráče: Ujistěte se, že hráč existuje v playerData.js
  • Chyba syntaxe: Použijte VS Code s rozšířením pro JavaScript
